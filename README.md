# API Test Studio

## Overview

API Test Studio is a lightweight REST API testing tool built with Go and Vue.js. It allows developers to send HTTP requests, organize API collections, and review request history through a clean web interface.

## Features

- JWT Authentication
- Collections
- Request History
- JSON Viewer
- REST API Testing

## Tech Stack

Go
Gin
Vue 3
SQLite
TailwindCSS
